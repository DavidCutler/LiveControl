Getting Started

Before you can begin editing airports for LiveControl, a few things need to be done. Please note, you will need a PC/Mac in order to edit airports.

Create a GitHub account and fork the repo. Clone the repo. A photo editor software (Photoshop or Gimp preferred)

  • Create a GitHub account, by entering your details on the GitHub website. Remember these details, they'll come in handly later!
  • Go to the Airports Repository, and select the 'Fork' button in the top right corner. This will make a copy of the airports under your account.
  • Download the GitHub Desktop app.
  • Login to the GitHub app, then tap the '+' button on the top left corner, select the 'Clone' tab, and select "LiveControl". Save this to somewhere you'll remember later.
  • Open your photo editor, and select the folder in which you cloned the repo from.
  • Import the image and begin editing!
  • Be sure to follow the color code rules: Red - Buildings Light Teal - Main Terminals Dark Teal - Movement areas Green - Grass areas Black - Everything outside the airport

Please check the Issue Tracker before starting your airport. Make sure it isn't already in progress, and if it isn't, please create a new issue.

Do's and Don'ts

Please read this section carefully, it will explain the main reasons as to why we will reject an airport submission.

DO

  • Have accurate colors for the airport
  • Make sure the airport hasn't already been edited
  • Export the file as a .png
  • The file name should be the last three digits of the ICAO code in lowercase and the word full EX: KLAX = laxfull

DON'T

  • Add text to the airport
  • Create random colors and implement them
  • Remember to color everything!
  • Make up an airport - it HAS to be a real airport

So you've just spent some time making an airport awesome... What next?

You need to send the changes you have made to us in order for them to be included in a future version of LiveControl! (your changes won't be instantly available for download in LiveConnect.

  1. In your photo editor, export the file as a .png (Highest quality you can possibly get!)
  2. Save the file.
  3. Open up an email client (such as Gmail).
  4. Send an email to ControlLive@gmail.com.
